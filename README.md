# OOPS Banner App

A Java console application that prints an "OOPS" banner using various OOP concepts.

## Use Cases

- **UC1**: Print OOPS to console
- **UC2**: Banner using string concatenation (+)
- **UC3**: Refactor banner using String.join()
- **UC4**: Render OOPS banner using Array and Loop
- **UC5**: Inline array initialization using String.join()
- **UC6**: Render OOPS banner using Static Function
- **UC7**: Refactor banner using InnerClass
- **UC8**: Use Map for character patterns and render via function

## How to Run

```bash
javac OOPSBannerApp.java
java OOPSBannerApp
```
